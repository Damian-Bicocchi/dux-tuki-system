Función: Poblar la base de datos con datos iniciales o de prueba (como el usuario administrador).
Qué va aquí: Scripts que insertan registros si no existen. Se ejecutan una sola vez al iniciar la aplicación o mediante comandos.
Ejemplo: admin.seeder.js (crea usuario admin con contraseña hasheada), categorias.seeder.js (insertar categorías por defecto).
